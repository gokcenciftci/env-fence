# Security Policy

## Supported Versions

Only the latest version of `EnvFence` is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not report it publicly. Instead, please report it directly by contacting the project maintainer. We aim to address security reports promptly.
